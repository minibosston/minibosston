
# Full-Stack Developer

Я full-stack разработчик. Основной стек у меня вокруг JavaScript и TypeScript: пишу frontend, backend, проектирую API, работаю с базами данных и собираю проекты так, чтобы их было удобно поддерживать и развивать.

Основной фокус держу на React, Vue, Next.js, Node.js, NestJS, PostgreSQL и Docker. Python неавижу всем сердцем и душой.
