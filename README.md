## About Me

Я full-stack разработчик, который любит собирать проекты целиком: от интерфейса и клиентской логики до API, базы данных и деплоя.

Работаю в основном в экосистеме JavaScript и TypeScript. Мне ближе понятные технологии, адекватная архитектура и код, который можно спокойно поддерживать и развивать, а не переписывать каждые два месяца.

Нравится, когда продукт собран аккуратно: быстрый frontend, внятный backend, нормальная структура проекта и предсказуемое поведение на каждом уровне.

Python вообще не мой инструмент. Основной стек и все ключевые задачи я сознательно строю вокруг JavaScript/TypeScript, потому что именно в этой среде работаю быстрее, увереннее и качественнее.
